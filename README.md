# BU EC_327: Introduction to Software Engineering (C++)

This repository contains all of my coursework, personal notes, and programming assignments for Boston University's EC_327 course as I learn C++.

## 📂 Repository Structure

* **`/C++`**: Contains my personal notes and in-class practice code.
    * `note1_intro.md`, `note2_loop_function.md`, etc.: My theoretical notes on C++ concepts.
    * `/BasicExamples`: Simple scripts testing out casting, loops, and logic.
    * `/ArrayCode`, `/PointerCode`: Practice files focusing on memory management and data structures.
* **`/Programming Assignment`**: Contains my assignments 

## 🚀 How to Run the Code

Most of the files in this repository are standalone C++ programs. You can compile and run them using `g++` in your terminal. 

For example, to run `Q5.cpp` from PA1:

```bash
# Navigate to the directory
cd "Programming Assignment/PA1"

# Compile the code
g++ -o q5 Q5.cpp

# Run the executable
./q5
```
